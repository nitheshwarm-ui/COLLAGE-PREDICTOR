# College Predictor System

## Project Title

**College Predictor System Using Student Academic Data**

## Objective

The main objective of this project is to help students predict suitable colleges based on their entrance exam scores, cutoff marks, category, and preferred branch. The system provides college recommendations and simplifies the admission decision-making process.

## Modules

### 1. User Registration and Login

* Student account creation
* Secure login authentication
* Profile management

### 2. Student Data Entry

* Enter entrance exam score
* Select category (General, OBC, SC, ST, etc.)
* Choose preferred course/branch
* Input location preferences

### 3. College Database Management

* Store college details
* Maintain cutoff marks
* Update seat availability
* Manage college rankings

### 4. Prediction Engine

* Analyze student score and preferences
* Compare with historical cutoff data
* Generate eligible college list

### 5. Recommendation Module

* Display suitable colleges
* Rank colleges based on eligibility
* Provide admission probability

### 6. Search and Filter Module

* Search colleges by name
* Filter by location, course, and ranking
* Sort results based on user preferences

### 7. Report Generation

* Generate prediction reports
* Download recommendation list
* Export results as PDF

## Technologies Used

* Frontend: HTML, CSS, JavaScript
* Backend: Python / Java / PHP
* Database: MySQL
* Tools: VS Code, XAMPP

## Expected Outcome

The system will provide accurate college predictions and help students make informed decisions during the admission process, reducing time and effort in searching for suitable colleges.
